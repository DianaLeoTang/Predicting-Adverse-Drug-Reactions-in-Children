# 基于共识衍生稀缺数据的儿童药物不良反应机器学习预测
项目概述

本项目旨在研究儿童药物不良反应的预测模型，利用计算生物学和机器学习技术来提高药物安全性和有效性。我们的目标是为临床实践提供数据驱动的支持，以优化儿童用药。

# 儿童药物不良反应数据集

本数据集伴随以下手稿：

**"基于共识衍生稀缺数据的儿童药物不良反应机器学习预测"**  
*Communications Chemistry* - 原则上已接受

## 数据集描述

本数据集包含专门为儿童人群整理的分子结构（SMILES格式）及其相关的药物不良反应（ADRs）。数据采用共识策略衍生，以解决儿童药物警戒数据固有的稀缺性问题。

## 文件结构

`final_dataset.csv`: 包含SMILES表示和标记的ADR任务的主数据集

**格式：**
第一列：分子结构的SMILES表示法
其余列：各个ADR任务的二进制标签（1 = 阳性，0 = 阴性，空 = 未标记）

## 生物指纹

对于本研究中使用的Biofeat，我们采用了**Chemical Checker**方法。由于模型体积较大，本仓库仅提供将SMILES转换为生物指纹的代码。详细的实现和预训练模型，请参考：

**https://github.com/ersilia-os/eos4u6p**

## 局限性

用户在使用本数据集时应注意以下局限性：

1. **数据稀缺性**：与成人人群相比，儿童ADR数据本身有限，导致某些ADR的样本量较小。

2. **基于共识的标记**：数据集采用共识策略以最大化数据效用，这可能会在不同数据源之间引入聚合偏差。

3. **报告偏差**：与所有药物警戒数据一样，本数据集可能受到漏报、选择性报告以及自发性不良事件报告系统固有的其他偏差的影响。

4. **年龄组异质性**：儿童人群涵盖不同的发育阶段（新生儿、婴儿、儿童、青少年），可能无法均匀代表。

5. **结构表示**：基于SMILES的分子表示可能无法完全捕获与生物活性相关的立体化学和3D结构特征。
